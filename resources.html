Resources for Building Your Portfolio

Follow these steps and use these tools to create a clean, professional portfolio – even from your mobile phone.

Step-by-Step: Create a Portfolio with GitHub Pages

  1. Create or log in to your GitHub account.
  2. Create a new repository named yourusername.github.io.
  3. Add an index.html file with your portfolio content.
  4. Go to Settings → Pages → select branch main and root folder.
  5. Your site will be live at https://yourusername.github.io.

What to Include in Your Portfolio

  • Clear headline (e.g., MCA 2025 | Domain | Open to Internships).
  • Short bio: 2–3 lines about your interests and strengths.
  • Projects section: name, tech stack, short description, GitHub link.
  • Skills section: languages, frameworks, tools.
  • Contact section: email and optionally LinkedIn.

Tips for Using Mobile Only

  • Use a mobile code editor app to write HTML/CSS comfortably.
  • Copy–paste final code into GitHub using the browser.
  • Keep layouts simple and test on your phone screen.
  • Update your portfolio regularly as you complete new projects.
